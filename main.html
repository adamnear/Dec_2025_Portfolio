Project Summary: Places of the World (Python for QGIS)

Places of the World is a custom PyQGIS tool designed to automate spatial demographic analysis. The program allows users to interactively select global locations within QGIS and generate a structured geospatial report based on Esri shapefile attributes.

Key Technical Features:

  • Spatial Data Integration: Developed a workflow to ingest point features and overlay them with OpenStreetMap (OSM) basemaps via the QuickMapServices plugin.
  • Interactive Selection: Built the script to recognize user-defined spatial selections (Polygon Tool). It features a "Default Selection" logic that processes the entire global dataset if no specific area is isolated.
  • Quadrant Analysis Logic: Programmed a spatial categorization engine that sorts points into the four global hemispheres (NE, NW, SE, SW) based on coordinate geometry.
  • Dynamic Reporting: Automated the generation of .txt reports that calculate:
    • Total location counts and aggregate populations per hemisphere.
    • Statistical extremes (identifying the highest and lowest population centers).
    • Complex edge-case handling for population ties, listing all relevant place names.
  • User Input Handling: Implemented a prompt-based system for custom directory and file naming, including automated file path string formatting.

Technical Takeaways: This project demonstrates proficiency in PyQGIS scripting, spatial attribute manipulation, and automated reporting. Future iterations aim to improve data visualization by outputting results in tabular formats and expanding the geographic attribute dataset.

Orthorectification Accuracy Assessment

Precision Orthorectification: Aerial Imagery Accuracy Assessment
This project focused on the high-resolution orthorectification of raw aerial photography using PCI Geomatica (Catalyst). By integrating a Digital Elevation Model (DEM) and georeferenced reference data, I transformed raw flightline imagery into a map-accurate product.

Key Technical Workflows:

  • Source Data Management: Utilized a KML air photo index to select and isolate specific flightline imagery from the Annapolis Royal dataset.
  • Orthorectification Process: Corrected geometric distortions by tying raw imagery to a 1m Digital Elevation Model (DEM) and a 25cm georeferenced base raster.
  • Rigorous Accuracy Assessment: Collected Ground Control Points (GCPs) and independent Check Points (CPs) across the scene to validate spatial integrity.
  • Error Optimization: Achieved a sub-pixel RMS error of less than 25cm, meeting strict project specifications for UTM Zone 20 NAD83 coordinate accuracy.
  • Visual QA/QC: Conducted a final comparative analysis between the output ortho-image and reference datasets to ensure seamless alignment and feature consistency.
